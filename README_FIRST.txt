FASTEST FIX (Choose one):

1. WINDOWS USERS - Double-click: diagnose.bat
   This will automatically test everything and guide you.

2. ALL USERS - Open in browser: test-simple.html
   This will tell you if JavaScript is working or not.

3. ENABLE JAVASCRIPT:
   
   Chrome/Edge:
   - Click lock icon in address bar
   - Click "Site settings"
   - Set JavaScript to "Allow"
   - Refresh page
   
   Firefox:
   - Type in address bar: about:config
   - Search: javascript.enabled
   - Make sure it's: true
   
   Safari:
   - Safari menu > Preferences > Security
   - Check "Enable JavaScript"

STILL NOT WORKING?

Try opening with a local server:

  Open Terminal in this folder
  Run: python -m http.server 8000
  Then open browser to: http://localhost:8000
